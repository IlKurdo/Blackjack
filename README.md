# Blackjack

## Come giocare

1. Avvia il programma.
2. Inserisci l'importo che vuoi puntare.
3. Il programma distribuirà due carte a te e due al banco.
4. Il tuo obiettivo è ottenere un punteggio il più vicino possibile a 21 senza superarlo.
5. Puoi scegliere di "chiedere" (hit) un'altra carta o di "stare" (stand) con il tuo punteggio attuale.
6. Se il tuo punteggio supera 21, perdi automaticamente.
7. Se il banco supera 21, vinci.
8. Se nessuno dei due supera 21, vince chi ha il punteggio più alto.

## Regole del gioco

- Le carte numerate valgono il loro valore nominale (2-10).
- L'Asso vale 1 o 11, a seconda di quale sia più vantaggioso.
- Un "Blackjack" è ottenuto con un Asso e una carta da 10 punti (punteggio totale 21).

## Come avviare il gioco

1. Assicurati di avere Python installato.
2. Salva il codice in un file chiamato `blackjack.py`.
3. Apri un terminale o prompt dei comandi.
4. Naviga nella directory dove hai salvato il file.
5. Esegui il comando: `python blackjack.py`

Per avviare la verisione **Android** serve un emulatore (Android Studio) e selezionando solo la cartella **AppAndroid**
